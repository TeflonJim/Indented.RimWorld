# Set-RWGamePath

## SYNOPSIS
Set the path to RimWorld if it cannot be discovered.

## SYNTAX

```
Set-RWGamePath [[-Path] <String>] [<CommonParameters>]
```

## DESCRIPTION
The path to the RimWorld game is used to discover the mods installed into the games directory.
