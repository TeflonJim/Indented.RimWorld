# Edit-RWModProfile

## SYNOPSIS
Opens a mod profile in a text editor.

## SYNTAX

```
Edit-RWModProfile [[-ProfileName] <String>] [<CommonParameters>]
```

## DESCRIPTION
Opens a mod profile, by name, in a text editor.
