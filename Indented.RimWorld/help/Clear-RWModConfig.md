# Clear-RWModConfig

## SYNOPSIS
Clears the mod load list.

## SYNTAX

```
Clear-RWModConfig [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Clears all mods except Core from the ActiveMods list in ModsConfig.xml.
